Procédure d'installation sur un poste local:
Simplement cloner le repo sur le poste.
Même si on run en local, l'App fonctionne avec l'api netlify. Si on veut utiliser
l'api local, il faudrait changer le lien de chaque requête à l'api fait dans l'app puisque
je n'ai pas utilisé de variable d'état.

Description de l'app:
simple application qui sert à liste différentes informations sur différents joueurs dans
le jeu minecraft. Les infos qu'on peut obtenir sont le nom du joueur, la version sur laquelle 
le joueur joue, si il est en mode hardcore ou non, son inventaire et ses succès. On peut
ajouter un joueur, on peut modifier un joueur, on peut supprimer un joueur, on peut lister
tout les joueurs, on peut chercher les joueurs selon le nom et selon la version du jeu.
